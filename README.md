# babulfish

> A library that provides translation of text on your page using client-side compute. Perfect for static web-sites.

[![NPM](https://img.shields.io/npm/v/babulfish.svg)](https://www.npmjs.com/package/babulfish) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save babulfish
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'babulfish'
import 'babulfish/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
