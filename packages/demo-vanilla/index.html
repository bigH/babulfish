babulfish Vanilla Demo

Direct @babulfish/core integration, no wrapper.

This page calls createBabulfish() directly and wires the DOM contract by hand: multiple roots, structuredText, DOM-only outputTransform, preserve/skip rules, restore, and RTL.

Root One

DOM roots, preserve, and skip

This article is one configured DOM root. babulfish, TranslateGemma, and WebGPU stay exact because the demo uses preserve.matchers before the DOM write.

The toolbar and status card stay outside the roots, so they remain stable while only the configured content translates.

SKU-2048 stays exact because this demo's shouldSkip rule leaves SKU-prefixed text alone.

Structured text keeps inline emphasis, links, and line breaks
while npm run demo stays opaque.
